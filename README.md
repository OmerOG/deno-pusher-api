# Deno Pusher API

This is a demo for a simple push api at scale using Deno + Hono + Socket.IO.

### Installation

Install Deno using Windows Powershell:
```powershell
irm https://deno.land/install.ps1 | iex
```

Enter cloned project and resolve dependencies:
```powershell
deno install
```

To run the project:
```powershell
deno run dev
```
